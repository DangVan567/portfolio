# Portfolio cá nhân - Nguyễn Văn Đăng

Website tĩnh gồm 3 phần:

- Giới thiệu
- Dự án học tập
- Tổng kết

## Cách xem trên máy

Mở file `index.html` bằng trình duyệt.

## Cách thêm file bài tập

Đặt các file PDF vào thư mục `assets/files/` đúng tên:

- `Tuan1_CNS.pdf`
- `Tuan_2_cns.pdf`
- `Tuan_3_cns.pdf`
- `Tuan_4_cns.pdf`
- `Tuan_5_cns.pdf`
- `Tuan_6_cns.pdf`

## Cách lấy link nộp

1. Tạo repository GitHub.
2. Upload các file website lên repository.
3. Vào `Settings` -> `Pages`.
4. Chọn `Deploy from a branch`.
5. Chọn branch `main`, folder `/root`, rồi lưu.
6. Nộp link dạng `https://ten-github.github.io/ten-repo/`.
